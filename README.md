# Breadboard

## Work in Progress. Still in early (re)development. 


> AI Media Browser

Browse, search, and manage all AI generated images and videos on your machine, in one place. 

This is an updated fork of [cocktailpeanut's Breadboard](https://github.com/cocktailpeanut/breadboard) with enhanced video support and additional features.

## Features

### Image Library
1. **Browse:** Browse images and their extracted metadata in one place
2. **Search:** Quickly search images by prompt and metadata
3. **Manage:** Select and bulk delete files, drag and drop to any other app for seamless integrated workflows
4. **Tag & Organize:** Add custom tags and favorite images for easy filtering
5. **Popout:** Open images in separate resizable windows with full controls

### Video Library (New)
1. **Browse:** Scan and index video collections
2. **Preview:** Hover to preview, click to expand
3. **Popout:** Open videos in separate resizable windows
4. **Lazy Loading:** Videos load on-demand as you scroll for better performance

# Supported File Formats

## Images
Supports common image formats:
- PNG, JPG, JPEG, WEBP, GIF, BMP, TIFF, TIF

Supports metadata extraction from files generated by:
- **Automatic1111** (and derivatives)
- **InvokeAI**
- **ComfyUI**

Designed with extensibility in mind and can support additional formats in the future.

## Videos
Supports common video formats:
- MP4, MOV, AVI, MKV, WEBM, M4V, FLV, WMV, 3GP, OGV

# Installation

## Prerequisites
- Node.js (for building from source)
- Python with `uv` package manager (for better-sqlite3 prebuilt binaries)

## Install from Source

1. Clone the repository:
```bash
git clone https://github.com/ai-anchorite/breadboard.git
cd breadboard/app
```

2. Install dependencies:
```bash
npm install
```

3. Start the app:
```bash
npm start
```

## Build Distributable

```bash
npm run dist
```

<br>

# Usage

## Image Library
1. Click the home icon to access the image library
2. Click "Settings" and "Edit" to add folders containing AI-generated images
3. Browse, search, and manage your image collection
4. Click any image to expand and view contained metadata
5. Add tags, favorite images, or bulk delete unwanted files

## Video Library
1. Click the video icon to access the video library
2. Click "Select Folder" to scan a directory containing videos
3. Hover over videos to preview, click play for continuous loop
4. Click to expand inline with audio controls
5. Use arrow keys to navigate between videos
6. Click the popout icon to open in a separate window

<br>
